#X Crawl - DCSS Fork

## Deviations from Vanilla Crawl

 - Ogres have a +2 Maces & Flails Aptitude
 - Jiyva & Lugonu generate in the Temple
 - The Necromancer starting spellbook contains 'Bolt of Draining'
 - Hornets don't paralyse anymore
